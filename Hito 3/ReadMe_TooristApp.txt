	EJECUCIÓN
	=========
Para poder ejecutar este proyecto se deben seguir los siguientes pasos:

1.Desde la CMD de windows, entrar a la carpeta del repositorio ya clonado,
 llamada TooristApp: "\...\ICF232_202010_Grupo9\Hito 3\TooristApp".

2.Crear el entorno virtual con el suguiente comando:
 >virtualenv venv

3.Al crear el entorno virtual, entrar a la carpeta Scripts dentro de venv:
 "\...\ICF232_202010_Grupo9\Hito 3\TooristApp\venv\Scripts".

4.Al entrar a esta carpeta, activar el entorno virtual con el siguiente comando:
 >activate

5.Mover el archivo 'requirements.txt' ubicado en la carpeta 'Hito 3', dentro de
 la carpeta Scripts, en la que se debería encontrar actualmente.

6.Al realizar esta acción, egecutar el siguiente comando:
 >pip3 install -r requirements.txt 

7.Para iniciar el servidor debe volver a la carpeta llamada TooristApp:
 "\...\ICF232_202010_Grupo9\Hito 3\TooristApp", y ejecutar el siguiente comando:
 >python manage.py runserver

8.Desde su navegador, ingresar a la dirección indicada en la misma consola y
 ya puede comenzar a navegar por el sitio.
 ej :"Starting development server at http://127.0.0.1:8000/"
